# Literature Reviews

This directory stores reading notes and summaries for each paper on the reading list.

**Full reading list with priority order:** see `/drafts/reading_list.md`

---

## File Naming Convention

`[AUTHOR-YEAR]-[short-slug].md`

Examples:
- `meng-2022-rome.md`
- `hartvigsen-2023-grace.md`
- `nanda-2025-open-problems-mi.md`

---

## Summary Template

Copy the block below when creating a new note.

```markdown
# [Full Paper Title]

**Authors:**
**Year:**
**Venue:**
**arXiv / DOI:**
**Category:** (e.g. Model Editing / Mechanistic Interpretability / Code Evaluation)
**Priority:** (1 / 2 / 3)
**Status:** [ ] To read / [ ] In progress / [ ] Done

---

## One-line summary
*What does this paper do, in one sentence?*

## Core contribution
*What is new here that wasn't known before?*

## Method / approach
*How do they do it? Key technical details.*

## Key results
*What did they find? Include numbers where memorable.*

## Relevance to this project
*Why is this on the reading list? What does it give us?*

## Critical notes
*Weaknesses, limitations, things to question.*

## Key terms / concepts introduced
*Vocabulary that will appear in the proposal.*

## Quotes worth keeping
*Any direct quotes useful for the proposal or report.*

## Follow-up reading flagged by this paper
*Papers they cite that might be worth adding to the reading list.*
```
