##### dots vs. underscores 
`dog_vet` (or any plain name) creates a variable in the local scope — it lives in that function and disappears when the function ends.

`dog.vet` attaches to an object — it lives on that object and persists as long as the object does.

`.` is saying "put this on that object's namespace" rather than "put this in the local namespace".

##### Trace and TraceDict utils 
`Trace` and `TraceDict` are custom PyTorch hook wrappers that let you intercept a model's forward pass at any named layer — either to read activations out or to edit them mid-pass.

the core surgical instrument that ROME needs at every step.

1. During Causal Trace: 
	`Trace` does this for a single layer (e.g. [causal_trace.py:656](vscode-webview://1221ru4qjitr9cj998n91g55gi23o2ooo2t1ut1rcmhke0e7t7h4/rome/experiments/causal_trace.py#L656) grabs the embedding layer), while `TraceDict` does it for a whole list of layers simultaneously (e.g. [causal_trace.py:212](vscode-webview://1221ru4qjitr9cj998n91g55gi23o2ooo2t1ut1rcmhke0e7t7h4/rome/experiments/causal_trace.py#L212) captures all transformer blocks at once to build the causal effect heatmap).
