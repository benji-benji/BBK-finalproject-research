For open-source models around the 2 billion parameter mark that specialize specifically in code generation, here are the primary options:

- **CodeGemma 2B (Google):** Built specifically for code completion and generation. Optimized for fast inference and latency-sensitive environments.
    
- **Qwen2.5-Coder 1.5B or 3B (Alibaba):** Derived from the Qwen foundation but fine-tuned strictly on code datasets. Consistently tops small-parameter coding benchmarks.
    
- **DeepSeek-Coder 1.3B (DeepSeek):** Trained from scratch on a massive composition of code and code-related natural language. Includes a 16K context window for project-level code infilling.
