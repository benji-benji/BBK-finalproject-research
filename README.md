# Model Editing Techniques: Capacities and Limits

MSc Data Science Final Project — Birkbeck, University of London
Supervisor: Dr. Taolue Chen | Proposal due: 7 May 2026

---

## What This Project Is

Comparing three model editing methods (ROME, MEMIT, GRACE) on standard knowledge-editing tasks using GPT-2. Own implementations, not library wrappers. The motivating question: can model editing influence broader behaviour beyond isolated facts? Stage 1 establishes the foundation; Stage 2 (final report) extends to code models.
